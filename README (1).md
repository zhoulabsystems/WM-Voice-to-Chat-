# 💧 Water Meter Recorder

A mobile-friendly web app for recording water meter readings using voice or manual input. Works offline and saves data to your browser.

## Features
- 🎤 Voice recording (Word mode or Digit mode)
- ⌨️ Manual entry with numeric keyboard
- 💾 Auto-saves offline (browser storage)
- 📊 Export to Excel (.csv)
- 📄 Export to Word (.doc)
- Works on Android Chrome and iPhone Safari

## Live App
👉 [Open the app](https://YOUR-USERNAME.github.io/water-meter-recorder/)

> Replace `YOUR-USERNAME` with your actual GitHub username after deploying.

## How to deploy (GitHub Pages)
1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch and **/ (root)** folder
5. Click **Save** — your link will appear in a minute

## Usage
- Open the link on your phone in **Chrome** (Android) or **Safari** (iPhone)
- Tap the mic button and allow microphone access when prompted
- Use **Word mode** for spoken words, **Digit mode** for reading numbers digit by digit without spaces
- Tap **Add Entry** after each reading
- Tap **Save offline** to make sure data is preserved
- Use **Excel** or **Word** buttons to download your records
